# Excuse Generator 🎭

An AI-powered excuse generator that creates convincing, context-aware excuses for any situation — missing a birthday, blowing a deadline, skipping the gym, whatever you need.

## Features

- **Prompt-based generation** — describe your situation and get a tailored excuse
- **Tone options** — choose from Professional, Comical, Friendly, and more
- **Powered by OpenAI** — responses that actually sound human

## How to Run

1. Clone the repo
2. Copy `config.example.js` to `config.js` and add your OpenAI API key
3. Open `index.html` in your browser or use VS Code Live Server

## Setup

```javascript
// config.js
const OPENAI_API_KEY = "your-api-key-here";
```

> ⚠️ Never commit your real API key. `config.js` is gitignored.
